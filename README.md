## Run Locally

Clone the project

```bash
  git clone https://github.com/D-Aman-S/Regex_Checker_IEEE.git
```

Go to the project directory

```bash
  cd Regex_Checker_IEEE
```

Install dependencies

```bash
  npm install
```

Start the local development server

```bash
  npm run dev
```
