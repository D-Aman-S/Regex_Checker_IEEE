## Run Locally

Clone the project

```bash
  git clone https://github.com/schietegal/Regex-Tester
```

Go to the project directory

```bash
  cd regex-tester
```

Install dependencies

```bash
  npm install
```

Start the local development server

```bash
  npm run dev
```
